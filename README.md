# HCC Guideline Site

This folder is ready for static hosting such as GitHub Pages.

Files:

- `index.html` - landing page
- `bclc.html` - bedside BCLC quick interactive version
- `full.html` - full 2026 guideline interactive version

Recommended GitHub Pages flow:

1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. Enable GitHub Pages from the default branch root.
4. Open the generated URL.

These HTML files already contain embedded images, so no extra asset folder is required.
